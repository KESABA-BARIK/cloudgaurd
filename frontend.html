ℹ OPMonitor Algorithm
Two-Phase Baseline Inference
Phase 1 — Coarse Mining
Groups logs by (userId, action). Finds longest common resource prefix. Emits broad wildcard rules. Uses ROI-Greedy submodular optimization.
Phase 2 — Rule Refinement
Examines each wildcard rule. Detects sub-directory structure. Splits over-broad wildcards into tighter per-subdirectory rules. Eliminates unseen access paths.
ABT Structure
Encodes refined rules into Abstract Baseline Tree for O(depth) lookup. Supports positive, negative, and wildcard conditions.
